# 🗨️ Conversalheia

**Conversalheia** é um chat ao vivo simples, desenvolvido com WebSockets. Ele permite que múltiplos usuários se comuniquem em tempo real por meio de uma interface web.  

## 🚀 Tecnologias Utilizadas

### 🔧 Backend
- **JavaScript (Node.js)**
- **WebSocket (nativo)**

### 🎨 Frontend
- **HTML**
- **CSS**
- **JavaScript**
